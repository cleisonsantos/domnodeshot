# Política de Privacidade — DOMnodeshot

Última atualização: 2026-04-18

DOMnodeshot foi projetada para funcionar localmente no navegador.

## Coleta de dados
DOMnodeshot **não coleta** dados pessoais.

## Armazenamento de dados
DOMnodeshot **não armazena** dados pessoais em servidores próprios.

## Compartilhamento de dados
DOMnodeshot **não compartilha** dados pessoais com terceiros.

## Processamento local
A seleção de elementos, captura de imagem, geração de arquivo e cópia para clipboard ocorrem localmente no navegador do usuário.

## Permissões e finalidade
- `debugger`: capturar imagem completa de elemento fora do viewport.
- `downloads`: salvar PNG capturado.
- `clipboardWrite`: copiar imagem ou HTML para clipboard.
- `scripting`, `activeTab`, `tabs`: injeção e controle do modo de seleção na aba ativa.

## Segurança
Extensão não realiza transmissão de conteúdo de páginas para backend externo.
